# LogoHustle ‚Äì Professional Online Logo Maker

LogoHustle is a powerful, browser-based logo design tool built with Next.js, React, and SVG. It allows users to create professional brand identities instantly without requiring a signup.

## üöÄ Features

- **Advanced Canvas Control**: 
  - Customizable dimensions and corner radius.
  - Background options: Solid colors and linear gradients.
  - Pattern overlays: Grid, Graph, and Dots with adjustable opacity and scale.
  - Soft-light noise filter for texture.
- **Rich Element Library**:
  - **Icons**: Full access to the Lucide icon library.
  - **Text**: Advanced typography with support for 20+ stylish Google Fonts (Pacifico, Lobster, Bangers, etc.).
  - **Shapes**: Rectangles, Circles, Triangles, Stars, and Hexagons.
  - **Lines & Curves**: Straight lines, quadratic curves, and cubic S-curves.
  - **Uploads**: Import your own PNG, JPG, or SVG files.
- **Pro Editing Tools**:
  - **Transformations**: Drag-to-move, resize, and rotate.
  - **Flipping**: Instant horizontal and vertical mirroring.
  - **Snapping**: Smart alignment guides for precise positioning.
  - **Layer Management**: Reorder, rename, and select elements through a dedicated layers panel.
  - **Styling**: Control fill, stroke, opacity, and drop shadows.
- **High-Quality Export**:
  - **SVG**: Vector-perfect export with embedded Google Fonts.
  - **PNG**: High-resolution raster export (captures all styles and fonts).
- **Undo/Redo**: Full history tracking for all design actions.
- **Offline-Ready**: Local storage persistence ensures your work is saved automatically.
