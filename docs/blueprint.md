# **App Name**: LogoForge

## Core Features:

- Lucide Icon Library: Browse and select from a library of Lucide React icons to incorporate into the logo design.
- Image/SVG Import: Import custom image or SVG files to use as logo elements.
- Canvas Editor: A canvas-based editor for arranging and manipulating logo elements.
- Text Layers: Add and customize text layers with options for font family, size, weight, alignment, and color.
- Shape Creation and Manipulation: Insert different types of shapes and enable scaling and other manipulation for each of these shapes.
- Export Options: Export the logo in PNG, SVG, and favicon.ico formats, preserving transparency.
- State Persistence: Automatically save the entire editor state to localStorage and restore it on page load.

## Style Guidelines:

- Primary color: Deep blue-gray (#344054) for a professional and sophisticated feel.
- Background color: Dark gray (#1E293B) to provide a sleek and modern backdrop.
- Accent color: Teal (#2DD4BF) to highlight interactive elements and call-to-actions, inspired by Figma's UI.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral look and excellent readability.
- Use consistent and clean Lucide React icons throughout the interface.
- Utilize a clean and responsive layout based on shadcn/ui components. Prioritize a desktop-first approach but ensure mobile usability.
- Incorporate subtle transitions and animations for smooth interactions.
